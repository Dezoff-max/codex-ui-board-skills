# Skill Comparison Guide

Use this guide to choose the right Codex UI board skill for a product surface.

| Skill | Choose it when | Avoid it when |
| --- | --- | --- |
| `dual-theme-spec-board` | You need a structured product UI with broad component coverage, clear states, and a neutral SaaS/admin feel. | The product needs a highly expressive or branded visual style. |
| `liquid-glass-aurora-board` | You want a premium glass interface for AI tools, launchers, glossy dashboards, or modern settings surfaces. | The UI must be extremely dense, plain, or conservative. |
| `neo-glow-control-board` | You are building dense operational tools, control rooms, QA dashboards, or technical admin panels with strong state visibility. | The experience should feel calm, soft, or consumer-oriented. |
| `soft-neumorphic-dual-theme-board` | You want a tactile, calm workspace for productivity tools, internal apps, dashboards, or settings screens. | The interface needs maximum density or strict enterprise minimalism. |

## Quick Choices

- Start with `dual-theme-spec-board` for general SaaS, admin, and settings workflows.
- Start with `liquid-glass-aurora-board` for premium AI tools and visually polished dashboards.
- Start with `neo-glow-control-board` for technical systems where status and controls matter most.
- Start with `soft-neumorphic-dual-theme-board` for calmer productivity and internal tool surfaces.

## Selection Questions

1. Does the interface need broad component coverage and neutral product UI structure?
   Use `dual-theme-spec-board`.

2. Does the interface need a premium, translucent, modern visual layer?
   Use `liquid-glass-aurora-board`.

3. Does the interface need compact operational controls and strong active states?
   Use `neo-glow-control-board`.

4. Does the interface need tactile depth and a softer workspace feel?
   Use `soft-neumorphic-dual-theme-board`.
